体彩超级大乐透“亿元派奖”首期(13062期)于6月1日开奖，江苏省南通海安县06464站点力擒当期3注大奖，总奖金达到2438万元。该大奖是否由一人独中?为何又分两次出票?随着大奖得主张先生昨日来到江苏省体彩中心领奖，谜底才被一一被解开。

　　大乐透频出大奖令江苏海安沸腾

　　本次2438万元大奖出自江苏南通海安县角斜镇06464站点，其实就在前不久的5月27日，海安的另一站点已擒获大乐透547万元大奖，5天之后又一更大的奖项降临海安，不得不说该地彩民是如此的幸运。记者走近角斜镇06464站点，店内店外聚集了好多彩民，大家纷纷猜测到底谁这么幸运，猜测的同时还不忘买彩票，说沾沾喜气，说不定自己也能中个大奖。据大家推测，中奖彩民可能是海安本地人，平时一直看好体彩超级大乐透，几乎每期都进行购买，但每期投入的金额较小，因为销售员推荐大乐透在进行大派送活动，因此不少人多买了一些，没想到还真被选中了。大伙儿还打趣道，中奖者也许正在想这笔钱如何使用，也许正在琢磨如何去领奖……因为3注大奖并非出自1张彩票，因此是否为一人所中，还须等到中奖者领奖后，才能知晓。

　　三保镖陪同中奖者现身领奖

　　6月5日中午，随着中奖者老张在三位保镖的陪同下现身江苏省体彩中心兑奖大厅，谜团终被解开，2438万大奖的确是一人独中!有意思的是，领奖前一保镖“护奖心急”，没等老张开口就声称自己是大奖得主。最终在兑奖大厅工作人员耐心地劝说下，老张这才站出来接受了记者的采访。据了解，老张也是位老彩民了，自打江苏发行体彩后就开始买彩票，大乐透上市后他开始把注意力转移到该玩法上，就像站点彩民猜测的那样“几乎每期都进行购买，但每期投入的金额较小”。老张每期都是固定投自编的5注大乐透号码，中奖那天同样也是这样，也正是因为大乐透开始“亿元派奖”，他才打破常规，在买完5注彩号之后，又返回站点，将自己非常喜欢的1注号码又倍投了两倍，结果这才有了一个站点中3注大奖并且分两次出票的神奇中奖。

　　大奖号码源自中奖者生日号

　　什么样的号码驱使老张又倍投了两倍呢?大乐透13062期的中奖号码为“01 05 08 0912+09 12”，这注彩号是老张多年来一直精心投注的号码。前区的前三个号码“01 05 08”谐音就是“要我发”，而前区和后区都有的“09 12”这组彩号恰好是老张的生日号。“我几乎每期都投这注号码，因为前后区都有我的生日号，叫起来顺口又吉祥，其他的号码其实我都换过，唯独这注没有换。”说到中奖，老张原本略显紧张的情绪不仅舒缓了很多，多少还带了一份得意。再看看中奖彩票，第一张在6月1日下午14点19分购买了5注单式，第3注就是他的幸运号，第二张则在6月1日下午14点36分购买，单独将这注幸运号码倍投2倍。两张票仅间隔17分钟，总共花费14元，便独中3注大奖，总奖金达到2438万元。
