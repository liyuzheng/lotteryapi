据新华社电法国电视2台13日晚间报道说，法国一名彩民独中价值1.69亿欧元(约合人民币13亿元)的“百万欧元乐透”大奖，打破这一博彩项目法国彩金纪录。

　　这位彩民，是分散在欧洲9个国家的4000万玩家中唯一选对了全部7个数字的人。据测算，这种幸运事件发生的概率极低，仅为一点一七亿分之一。

　　目前，法国博彩公司为防止有人冒领奖金，对获奖者的身份进行保密，拒绝透露该注彩票在何处销售。作为这一期开奖的唯一赢家，那名彩民所获奖金足够买下将近4吨黄金。

　　根据游戏规则，获奖者必须在60天时间内到彩票公司领奖。如果此人身份经多次核实无误，法国博彩公司将负担获奖者前来领奖的路费和住宿费，在极为保密的状态下交付奖金支票。如果过期无人领奖，则记入下期奖金之中。

　　依据《挑战》杂志财富榜，领奖后，这名幸运儿将列法国财富榜第221位。

　　法新社报道，这名中奖者打破“百万欧元乐透”法国彩金纪录，比前一名最高奖得主多700万欧元(890万美元)。先前纪录保持者是住在卡尔瓦多斯省东北部的一名男子。

　　本次奖金数是2004年“百万欧元乐透”创立以来的第三大巨奖。2012年8月与2011年7月，有英国彩民曾经获得1.9亿与1.8亿欧元奖金。该奖项创立8年来，共有260人获奖，其中法国人72名、西班牙人58名、英国人57人。
