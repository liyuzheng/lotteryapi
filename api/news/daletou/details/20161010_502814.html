“十一”国庆黄金周，体彩大乐透头奖欢乐大井喷!10月5日，体彩大乐透第16117期开奖，全国送出的5注767万余元头奖，花落上海、浙江、福建、河南和湖南五地。其中，上海彩民通过一张10元5注单式票，擒得一等奖1注、二等奖1注，单票奖金为787万元。10月8日中午，中奖彩民谢先生在其爱人的陪同下火速完成兑奖。

　　中奖之后的小两口心情大好，采访全程都展露着灿烂的笑容。两位80后夫妻来自江苏，大学毕业起在上海打拼。他们表示：“我们是农村出来的孩子，在上海打拼了十多年，一直想通过自己的奋斗在上海拥有一套自己的房子，这次体彩的大奖会帮助我们更快地实现理想。”

　　中奖者谢先生是大乐透的资深彩民，购彩有近十年时间，他说：“我第一次买大乐透是机选，我把那组号码进行了几个数位上的改动，就形成了这张10元5注单式票，随后开始坚持守号。与此同时，我每期还另外买10元机选票，也是5注单式，每期共花费20元，一直着保持这个习惯。可以说，能中奖就是贵在这份坚持吧!”

　　当天晚间得知中奖小两口激动得一夜没合眼

　　10月5日傍晚，谢先生赶赴位于浦东新区莲溪路550弄8号的12423网点购买了该注彩票，晚间开奖之后，谢先生便在手机上查询了开奖结果。回忆起当时的情况，他至今难以平复心情：“当时我真是看了好几遍才敢相信，一晚上基本都没睡着，我老婆也相当激动，两个人都感觉不可思议。”

　　成为了彩民们都羡慕的那个大奖幸运儿，谢先生却坦言，会时常告诫自己，心态要放平稳。他说：“我们老家的家庭条件艰苦，今天我们得到了幸运，一方面可以多多支援老家的亲戚，一方面改善改善自己的生活。但是生活不会改变，我们仍然要踏踏实实的工作，以前怎么生活，如今还是怎么生活。”说着，他拿出一张纸条，笑言：“你看，今天又是大乐透的开奖日，我重新编了一组守号的号码，今天晚上会继续投注。”
