周五001 日职 神户胜利船VS大阪钢巴

　　比赛时间：08-02 18:00

　　日本联赛第21轮开战，赛程进入下半程，争冠与保级的争夺愈发激烈。神户排名第十五，大阪钢巴排名第十一，看积分形势都有些靠后，大阪钢巴算是中游偏下，神户则是典型的中下游球队，神户只高出降级区3分，看起来抢分动力更足一些。

　　神户主场3胜2平4负，主场战绩和排名一样，并不理想。9个主场打进16球丢16球，打法异常开放，拥有比利亚、伊涅斯塔和波多尔斯基他们本赛季遭遇财政危机，目前人心有点散，好在比利亚老当益壮，至今打进10球，在射手榜上并列第二。

　　大阪钢巴客场3胜3平4负，客场战绩也是非常一般，10个客场打进13球丢16球，也属于不保守的球队，本场面对明星云集的神户，大阪钢巴很有掰一掰手腕的想法，一场对攻战呼之欲出。从两队的攻防实力来看，神户取得进球不是问题，进球数1-2个;大阪钢巴取得进球也是大概率事件，进球数同样1-2个。

　　两队本赛季首次交手就打出大比分，大阪钢巴主场3-4输给神户，比利亚和波多尔斯基都取得了进球。近况，神户遭遇联赛三连败，每场至少丢掉2球，状态低迷;大阪钢巴最近三轮胜平负各一场，谈不上出色，但是比神户好得多，每场比赛都有进球，进攻还算是有保障。

　　既然本场极有可能沦为对攻战，那么我们可以从大小球入手，本场大小球开出2.75低水，目前已经升到3球中水，大球强势符合基本面，本场大球值得期待。

　　初始指数神户让平半中高水，目前变化不大，神户排名靠后、状态糟糕，能在主场让球本身就是对他们的支持。平局指数在3.60+超高位置，对攻战确实不利于打出平局，这种高平局是真实的。客胜有刻意的拉低，实际上以大阪钢巴的客场能力也不足以赢球。本场主胜首选，比分考虑3-2。

　　竞彩推荐：主胜

　　进球推荐：3、4、5

　　( 文章来源：五星冷刀 )
