3月12日，距大连市马年岁尾中出的3318万元双色球大奖开奖日已到了第29天上，就在所有彩民和大连市福彩中心工作人员企盼大奖有人认领、近乎绝望的时候，中奖者却在当天上午9点半神奇出现了。29天，对常人来说就是平淡无奇的每个日子，而对于意外得到3000多万元巨款，这笔巨款又没攥在自己手里的人来说，该是怎样度过的?更让所有人不解的是，知道中奖、彩票没丢，为何过了这么久才来兑奖?大奖得主柳先生(化名)一句轻描淡写的回答，让所有人大跌眼镜：“没啥，就是太忙了，没时间来。”

　　彩票站业主透露：近日就去兑奖

　　眼看着兑奖最后期限越来越近，为了不让弃奖事件发生，大连市福彩中心一直在想方设法寻找大奖得主。就在3月10日与大奖中出站点第21020795号投注站业主王先生联系时，对方却透露了一个让人惊喜的讯息：他听站点所在小区住户说，认识大奖得主，其早就知道自己中奖，只是春节回老家过年了，近日就回来，马上就会去兑奖。王先生还安慰市福彩中心工作人员：“放心，不会弃奖。”他还透露，春节后给市福彩中心打电话咨询兑奖规定的正是中奖者的妻子。

　　中奖者：开奖第二天即知中奖

　　大奖得主柳先生的出现印证了彩票站业主的说法，但他却说这段时间自己哪也没去，就在大连，之所以一直没来兑奖只是因为太忙。至于为什么这么忙，以至于连3000多万元巨款都顾不上领，柳先生笑着说：“不便透露。”

　　据柳先生介绍，当期彩票开奖的第二天早晨他就已经获知自己中奖了。他订阅了《大连晚报》，每天都会从报纸上查看彩票开奖号码。当他发现当期双色球开奖号码与自己当期选的号码一个不差时，虽经反复核对，还是不能相信自己的眼睛，还让妻子帮着核对了好几遍。

　　柳先生说，自从获知中大奖后的将近一个月里，他们一家工作、生活一切照旧，而那张价值3000多万元的中奖彩票就安静地躺在家中的钱夹子里，因为家里没外人来，所以不担心会丢。对于兑奖期限，他说原先只是隐隐约约知道一点，但不是很确定，就让妻子给市福彩中心打电话确认了一下。由于心中有数，确定不会错过兑奖时间，直到自己终于有时间了，柳先生才不慌不忙来到市福彩中心兑奖。不过没真正见到钱的那些天，柳先生时不时地会怀疑这事是否是真实的，他透露说，就在临来兑奖前，他还拿着那张彩票又跟当期的开奖号码对了一遍。

　　如此罕见的心态和做法不能不让人称奇，但有人却提出，这么一大笔钱不抓紧存银行，利息得损失多少?柳先生对此满不在乎：“没多少，最近还降息了。”这真是有钱挡不住任性啊。

　　两张彩票：一张守号一张每期不同

　　说起买彩票的经历，柳先生话多起来。他和众多福利彩票忠实彩民一样，自从有了电脑彩票后就开始买了，原先是双色球、3D都买，后来只专注于双色球，期期不落，但每次最多不超过20元钱。他说自己就把彩票当做一种休闲方式，觉得乐在其中。不过他也实话实说：“要说不想中奖那是假话。”但10多年来中过的最大奖也就200元，他也不抱怨。

　　柳先生介绍自己买彩票比较有规律，他也研究走势图，研究报纸登的走势图。几年来，柳先生基本每期都买两张彩票，一张是5注单式票，另一张是1注号码5倍投注;5注单式票是守号投注，1注号码5倍投注的每期不同，此次中奖的就是那张1注号码5倍投注的彩票。

　　据柳先生回忆，在选2月12日开奖的双色球第2015019期号码时，他看着走势图，凭着感觉，首先认定了蓝球很有可能出“04”;在选红球时，他先确定了7个号码，其中有一组三连号，7个号码分别是“02、06、11、19、25、26、27”。他说，7个号码必定要杀掉1个，当时他有两个选择，或者杀掉“27”或者杀掉“02”，结果他最终杀掉了“27”，赢得了一等奖;假如杀掉“02”，那他只能中得5倍三等奖，奖金只有1.5万元，与3000多万元简直就是天壤之别。对此柳先生说：“这只能说我运气好啊。”

　　那张走势图上圈圈点点、7个号码最终杀掉一个“27”号的报纸被柳先生一直保存着，他准备作为纪念永久保存下去。

　　希望适当时机救助贫困

　　整整29天，不兑奖或许是在商量这笔钱怎么处置吧?可柳先生的回答同样让人大跌眼镜：“没什么特别的想法。”不过一家人还是有个大致的想法，上完税后这笔奖金拿到手里还有2600多万元，换一套大一点的房子是首选。他还特别提出想捐点钱给慈善机构，但他又打消了这个念头，打算从媒体上发现报道家庭遭遇不幸的，那时会为其进行捐助。柳先生还表示，家里亲戚中有的生活比较困难，平时都会不时地给予些接济，现在中大奖了，理所当然地会给他们一些资助。

　　奉劝彩民坚持就会有机会

　　谈到中奖感受，柳先生说作为老彩民，尽管买彩票十多年，此前中过的最大奖是200元，但他始终相信媒体对中大奖的宣传不是传说，开奖不会虚假，之所以人家中了自己没中是自己“运气没到”。这回通过自己中大奖的经历，更说明买彩票中大奖是真实的，只要坚持就能有机会。而一旦中了大奖就可以让生活得到改善，即使买了多年不中奖想到是为社会做贡献也会得到一种心理安慰。

　　795万元大奖仍没人认领

　　春节前我市连续中出的三个双色球一等奖，如今两个已经兑了奖，让人们在大大地松了一口气的同时，仍有一点牵挂，那就是还有一个大奖没人认领：紧随3318万元大奖之后中出的795万元大奖。截至3月12日，距开奖时间已过去26天，那个中奖者，你在哪里?

　　795万元大奖是在2月15日双色球第2015020期中出的，与3318万元大奖中出仅差3天，中奖彩票为一张机选4注号码单倍投注计8元投注额的彩票，购买时间为2015年2月14日10时23分49秒，购买地点为西岗区纪念街金源商场(纪念街北段入口处)的第21020524号投注站。

　　据524号福彩投注站销售员于女士回忆，在该站点的众多彩民中，仅有两人是经常机选双色球的，一个是位年轻小伙儿，一个是位女士。大奖中出后，那位女士又出现在彩票站，看样子不像是中奖者，而那位年轻小伙儿此后再未出现过，不知是回家过年没回来，还是其他什么原因。

　　于女士介绍说，除了获知中出大奖那天在彩票站门口贴出喜报、放了鞭炮外，他们后来还在门前挂了条幅，在站点内部双色球走势图上也张贴了中奖喜报，经过站点或到站点买彩票的人都会看到，即使中奖者原先不知自己中奖，看到这些讯息，自然就会知道了。但自大奖中出后却没有任何人向他们询问兑奖的有关事宜。

　　于女士说，他们很为中奖者担心，包括来买彩票的彩民也都议论纷纷，猜测谁是那位中奖者。有人说：“谁中的?还不赶紧去兑奖，光银行利息就损失多少啊?”也有人调侃：“要是我中的，我早就赶紧去兑了，夜长梦多啊。”

　　于女士表示，这位中奖者因为是机选，又赶上春节前，很有可能根本就不知自己中奖，或者是彩票找不到了，不能确定自己是否是中奖者。谁遇到这种事都会很懊恼，哪还肯露面?她说，如果那样的话，这个大奖恐怕就会弃奖了。她希望这样的结果最好不要发生，希望奇迹能够出现，就像3318万元大奖得主那样，给大家一个惊喜。她表示，自己也会再想办法提醒每个到站点来的彩民，大家帮忙寻找那位中奖者，离兑奖最后期限还有30多天，大家一齐喊他赶紧去兑奖。（大连福彩晓吾）
