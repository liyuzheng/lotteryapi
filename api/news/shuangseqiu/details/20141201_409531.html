11月20日晚，中国福利彩票双色球游戏进行第2014135期开奖，由于当期双色球头奖全国开出4注1250万元头奖，因此，当期开出的210注二等奖单注金额为14.7万多元。特别值得一提得是，当期河南省出了为超级二等奖得主――一位洛阳彩民在位于洛阳市涧西区嵩山路中段的编号为41030001号福彩投注站用一张132元的单式倍投彩票一举获得二等奖66注，总奖金971万元。就在大家热议这这位超级二等奖得主，并为他一号之差错失3亿大奖的同时，大奖得主张先生(化名)现身河南省福利彩票发行中心兑奖室领取了这笔大奖。

　　与家人一起前来领奖的张先生脸上看不出一点错失3亿元大奖的遗憾，“我就知道我没有中一等奖的命，所以一点也没有感到难过”张先生在表达了自己现在心情的同时还解密了他买这张彩票时离3亿元大奖更近的一幕情景。原来，张先生当期中奖的这注彩票是机选的，当时机选蓝号就是01，可他觉得01刚刚出过，所以就换成了05。联系到当期双色球兑奖后奖池3.32亿元的事实，张先生总结道：“我买66倍是对的，一旦中奖刚刚好可以打空奖池，但是我自己将号码改掉也是对的，因为我好像自买彩票以来就感觉自己不会中大奖，另外还有一点，我知道自己现在还没支配3亿元大奖的能力。”

　　张先生在领奖时讲，他买彩票有十多年了，福彩刚上市时他还在学校上学，当时就觉得这个挺好玩，那时他就会拿出两元钱或四元钱买上个一两注彩票搏一搏运气，也许是与彩票有缘，他刚买时就中了奖，虽然不多可也给他以后很长一段时间奠下了买彩票的底气“反正都是中的奖金，大不了买回去就完了，也不损失啥”。抱着这种放松心态的张先生买彩票越来越顺，从上学到上班，随着收入的增长，他买彩票的金额也在增长，可有一点却一直没变：基本上买彩票的钱都是之前中奖的奖金，保证买彩票不会影响到自己日常生活。

　　“心态都是历练出来的，经历多了，也就能够淡然面对这种错失大奖的遗憾了”。张先生在阐述自己心态历程的同时随口就给我们讲了几个小例子：有一回他在一段时间内一直守着一组号码投注，轮到一期开奖前本想买彩票的他想想离开奖还有一天就没有及时购买，可等到临开奖前，他却被工作上的事缠住，想着一期不买不会那么巧吧，结果第二天一看，一注到手的一等奖没了。还有一段时间，张先生喜欢跟着自己上期中奖的号码买，有一次他去投注站兑奖时本想在兑奖后继续守着买的，可投注站业主在兑完奖后随手将他的那张中奖彩票给扔到垃圾篓里，结果就是那组号码，第二天按照开奖结果可以中得三十八万多元奖金。

　　“随性而为的性格可能会让你错失大奖，可有时也会让你有意外收获”。张先生在给我们讲了他数次与大奖失之交臂的经历后，还给我们讲了个“拾”得大奖的例子：有一次，张先生买彩票进了一个福彩投注站，见一位老太太在选双色球号码，看见她选号码非常认真的样子，张先生想：买彩票总不中奖肯定很苦恼，专心选号一定也很希望别人得到别人的认同与尊重吧。于是张先生就与她商量可不可以用用她挑的双色球投注号码，在得到同意后，张先生一看是一注标准的单式票，就另外加了几个红号进去买了注复式投注票，结果哪位老太太中了个4+1，而加了几个红号买成复式的张先生却中了小一万的奖金。

　　“买彩票要根据自己的实力，不做问题彩民;对待奖金，更要看自己的心态，不能做金钱的奴隶”。张先生说自己在多年的购买彩票历程中，看过太多为了彩票孤注一掷的人，他很为他们着急，要知道，彩票不像股票，不中就是一张废纸了。“看了你们之前的一个报道，几位彩民追3D号码，追了大半年，中了几百万，要是不中的话，他们的心情可想而知啊。”所以对待彩票就要像对待自己日常生活中得一个正常爱好，可以中奖为目的，但不能执着。不中了，要乐观对待，无论投入多少，哪怕只中一个大奖也够了;中了讲，更要合理支配奖金，不能一中大奖就忘乎所以，像国外报道的很多大奖得主忘乎所以，最后把一件好事变成了坏事。最后，只要能保持一个好的心态，无论中奖与否，肯定都能从购买彩票中找到乐趣，能让自己的生活更鲜活起来。
