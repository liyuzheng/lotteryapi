　　马力当天晚上忍痛去ATM机取出3000块，取钱的时候，还有三个同伴跟着，监视自己，马力也没机会跑。取出钱后，回出租屋的路上，马力用200块买了不少好吃的，反正一个份额2800，还剩200块，要尽量先笼络人心，后面才有机会。

　　那些家伙，多少日子没吃到带点油水的了，马力把买来的东西分给大家，小李飞刀却要阻拦，说：“马力，你这是腐蚀大家的意志，我们的口号是，吃得苦中苦，方为人上人”。

　　马力说：“为人上人的目的是什么，不就是为了过得好点吗，现在过得好和以后过得好都一样”。

　　小李飞刀说：“怎么能一样，现在过得好，以后就过不好了，现在忍受了别人不能忍受的，以后才能发达”。

　　看来小李飞刀被洗脑的不轻，同住的几个被洗脑厉害的也站在小李飞刀一边，不过大多数人很多天没吃到好的，还是嘴馋，小李飞刀他们寡不敌众，也阻挡不了，只能自己不吃算了。

　　第二天，领导讲课的时候，马力递上前一天取出来的2800购买了份额，领导很高兴，说马力有前途。不过，领导又说：“只购买份额不行，还要拉动闲散资金，晚上你给朋友们打几个电话，也照顾下朋友，别只顾着自己发财”。

　　马力这一天都犯愁，给谁打电话呢，怎么说呢?别真的把朋友骗来了，自己以后怎么见人家?晚上吃完饭，那天要去马力手机的女孩找到马力，说领导让马力打电话，就把马力带到最小的那间房里，房里没有别人，桌子上有一部固定电话，女孩让马力自己打，她就站在一边听着。

　　马力无奈，拨通了尤凌的电话：“喂，尤凌吧，我是马力啊”。

　　两人寒暄几句，马力说：“尤凌，你上次不是说要我给你介绍男朋友吗，我这边有个男孩，人特好。你要不给公司请几天假来看看”?

　　马力故意乱说，尤凌那边高考已经结束了，听的云里雾里，自己才高中毕业，怎么会让马力介绍男朋友，再说什么向公司请假，自己还没工作啊。

　　马力接着说：“什么，你这几天赢了不少钱，不错不错。今晚的比赛怎么看?我给你说，这东西，大的原则一定要记住。我给你说，我这几年的经验总结就四句话，你听好了，记住了。传世宝典赢盘王、消极情绪不可有;勿把表象当真相、信盘口方能盈利。尤凌，这四句话，你好好琢磨琢磨，琢磨通了，以后盈利的机会还很多”。

　　“好了，先不说这个，还是说我给你介绍男朋友的事吧，怎么样，过段时间请假来看看”。

　　“什么，你要考虑考虑，好嘞，你先考虑两天，我再给你电话”。

　　马力和尤凌这样胡乱地说了一通，心里祈祷，尤凌一定要明白自己的意思，还有，千万别在qq群里把这事说出去，让小李飞刀看到了就坏了。不过还好，小李飞刀在这里也没有自由，一周只有一次机会去网吧上网忽悠网友。

　　马力想，一周，自己怎么也应该出去了吧。马力又给老庄打了电话，也是胡乱说了一通，说给老庄介绍个女朋友，老庄女儿都十几岁了，听得老庄糊里糊涂。之后又重复了他那句经验总结：传世宝典赢盘王、消极情绪不可有;勿把表象当真相、信盘口方能盈利。

　　第二天，马力在会上给领导报告了打电话的结果，说他就说给朋友介绍男(女)朋友，有几个朋友有意思，等过几天请好假，准备过来看看。

　　领导问了监视马力的女孩，女孩把当时的对话给领导大概复述了一边，说最后他们聊了聊分析比赛的事情，我也听不懂说的什么。领导问马力，马力说就是一些看盘的技巧，我这些年的经验总结，这个小李飞刀明白。

　　马力把那天的话偷偷改了两个字，念给小李飞刀听：奇世宝典赢盘王、负面情绪不可有;勿把表象当真相、信盘口方能盈利。小李飞刀也听不太明白，又觉得说不明白没面子，就说，这个是一些分析比赛常用的方法，以前马力也说过，我都懂。领导听了，说马力这个方法好，值得推广和大家来借鉴。

　　这两天，马力又取了一次3000块，和第一次一样，2800购买了一个份额，剩下200给大伙买了好吃的。第四天，马力给领导说：“我那张农行的存折上有20000元，不过农行的存折没有办卡，而且存了定期，只能自己拿身份证去柜台取。马力说，自己想把这20000元也买成份额，这样以后就赚得多点”。

　　领导想了想，最后决定把马力的身份证先给他，另外派两个信得过的陪着他去一次银行柜台。马力想，机会来了，这一次一定要逃出去。
