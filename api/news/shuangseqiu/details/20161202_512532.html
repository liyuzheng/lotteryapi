12月1日晚，中国福利彩票双色球游戏进行第2016141期开奖，这是2016年双色球9亿元大派奖活动启动后第十八期派奖。当期开出的红球为04、13、15、17、21、24，蓝球为15。

　　当期双色球开出头奖7注，其中4注为复式投注，单注奖金为1318万多元(含加奖500万元)，其中江苏1注，河南1注，海南1注，青海1注。另外3注为非复式投注，不参与派奖活动，单注奖金818万多元，这3注非复式投注一等奖花落3地，其中安徽1注，广东1注，云南1注，共3注。

　　当期双色球六等奖开出859万多注，其中384万多注为复式投注，参与派奖活动，最后中奖金额为每注10元(含加奖5元)。当期二等奖开出106注，单注金额26万多元。

　　当期红球号码大小比为3:3，三区比为1:4:1，奇偶比为4:2。当期红球开出一组奇连号13、15、17;一枚重复号17;一组同尾号04、24。蓝球则开出15。

　　双色球第2016141期全国销量为3.71亿多元。广东(不含深圳)当期双色球销量为3293万多元，高居第一，浙江以2932万多元的销量位列第二，江苏2213万多元排名第三，山东2141万多元排名第四，四川则以1802万多元排名第五。

　　计奖后，奖池金额为10.25亿元。本期6000万元一等奖特别奖派奖奖金派出2000万元，剩余4000万元，按本次派奖规则，剩余派奖奖金自动滚入至下期第2016142期，下期复式投注中得一等奖按中奖注数均分6000万元。本期计奖结束后，一等奖特别奖剩余派奖金额8000万元，六等奖特别奖剩余派奖金额为7879万多元，按照派奖规则，本期计奖结束后，双色球9亿元大派奖活动仅剩两期，请广大彩民把握机会。由于单注派奖奖金最高限额500万，下期彩民朋友将有机会4元中得1500万元。
