1月8日晚，中国福利彩票双色球游戏进行第2013004期开奖，当期井喷17注529万元(含加奖)头奖。本期黑龙江省共中出一等奖5注，据数据显示，5注头奖均被一人独揽，中奖彩票出自哈尔滨市道里区达道街48号第23010231号福彩投注站，这位幸运彩民仅凭一张自选10单式票，轻松拿下5注双色球头奖，共获得奖金2645万元(含加奖)。这是今年我省双色球中出的首注头奖，如此庞大的奖金，为龙江彩市在新年开了一个幸运、吉祥的好兆头。

　　红球偶数号码持续高开 奇偶比1:5高频开出

　　本期红球号码为06、10、16、20、27、32。号码和值为111;跨度26;三区比为2:2:2;质合比为0:6;奇偶比为1:5。本期开出2组同尾号：06、16和10、20;无连号开出。开出1枚重号：27;无跳号开出。

　　本期红球号码奇偶比为1:5，2013年以来的四期开奖中，该比值已经出现2次，近10期开奖该比值累计出现3次。据统计，双色球玩法2012年全年的154期开奖中，奇偶比1:5累计开出12次，占比7.79%。奇偶比1:5的理论开出概率为6.7%，小概率的开奖形态短期内高频开出实属诡异。[ 年终盘点：解密2012双色球号码神秘关系 ]

　　近5期开奖中，四次开出偶数占优组合、一次开出奇偶均衡组合。5期30枚开奖号码的奇偶比为9:21，近期奇数号码或多开回补。

　　本期6枚红球近期的开出情况统计如下：号码32近10期开出4次;号码16、27近10期均开出3次;号码06、10分别开出2次;号码20是10期内首次开出，开出前遗漏12期。

　　开17注529万元头奖 哈尔滨一人独揽5注

　　双色球第2013004期全国井喷17注头奖，单注奖金529万元(含加奖)，其中黑龙江中出5注，成为当期双色球游戏的最大赢家，其余12注被山东(3注)、海南(2注)、天津、内蒙古、辽宁、江苏、湖北、陕西、深圳彩民分享。

　　继去年龙江彩民收获47注双色球头奖的胜利果实之后，新年伊始，在双色球游戏仅开奖4期的情况下，龙江彩民就已经创下了辉煌的战绩。从1月1日至1月8日，双色球进行第2013001期、002期、003期、004期开奖，龙江彩民期间共揽获头奖5注，二等奖10注，可谓“弹无虚发”。尤其本期哈尔滨彩民一人独揽5注双色球头奖，获得2645万元大奖的喜讯，更是在全国彩市上独领风骚，令全国彩民瞩目。这次哈尔滨市彩民独揽的2645万元大奖以超出20万元的成绩，刷新了之前(双色球第2013001期)由广州市彩民创下的一人独揽头奖2625万元奖金的今年全国双色球最高奖金记录。

　　双色球奖池3556万元 2亿元派奖活动继续

　　第2013004期双色球全国销量为3.37亿多元。当期双色球井喷17注529(含加奖)万元头奖，下期双色球奖池金额为3556万元。相信在新的一年里，哈尔滨彩民的这份幸运会在龙江大地继续传递下去，龙江彩民在奉献爱心的同时，会收获更多的大奖。下期双色球游戏将继续派奖，彩民朋友有机会2元擒获1000万元。
