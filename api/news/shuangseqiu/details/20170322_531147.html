3月21日晚，中国福利彩票双色球游戏进行第2017032期开奖。当期双色球红球号码为05、08、15、24、27、31，蓝球为11。

　　头奖63注500万 花落19地

　　当期双色球头奖井喷63注，单注奖金为500万元，这63注一等奖分落19地。其中，北京2注,天津1注,河北3注,内蒙古2注,吉林2注,上海2注,江苏9注,浙江2注,福建1注,江西1注,山东2注,重庆1注,河南1注,湖南4注,广东19注,四川1注,贵州2注,云南6注,深圳2注,共63注;值得一提的是，广东揽19注一等奖，若为一人所中，则本期将产生9500万巨奖。当期中彩中心决定:从双色球调节基金拨出28804447.88元,弥补当期派奖奖金不足。

　　二等奖开出284注，单注金额6000元。其中，北京中出47注，排名第一;广东(不含深圳)中出36注，排名第二;浙江中出29注，排名第三;山东中出18注，排名第四;江苏、湖南各中出13注，并列排名第五。此外，其他地区二等奖均在13注以下。当期末等奖开出2175万多注。

　　当期红球号码大小比为3:3，三区比为2:1:3;奇偶比为4:2。其中，红球开出三枚斜连号05、15、27;一枚重复号27;一组同尾号05、15，蓝球开出11。

　　奖池6.09亿元

　　当期全国销量为3.46亿多元。广东(不含深圳)当期双色球销量为3071万多元，高居第一，浙江以2714万多元的销量位列第二，山东以2074万多元排名第三，江苏以1954万多元排名第四，四川则以1598万多元排名第五。

　　计奖后，双色球奖池金额为6.09亿多元，下期彩民朋友将有机会2元中得1000万元。
