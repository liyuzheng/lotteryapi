北京时间3月7日下午3时，广州双色球1.08亿得主现身领奖，距离他的中奖时间--3月5日21:30，仅过去了41个半小时!与2.48亿得主一起并列最迅速兑奖的亿元得主，他是个20多岁的小伙，号码已经守了大半年，在姐夫的陪同下完成兑奖。

　　昨日下午3时，广州1.08亿多元巨奖得主在姐夫的陪同下悄然来到广东省福利彩票[微博]发行中心，是中国亿元巨奖得主中最快现身的。(注：原文如此，体彩2.48亿得主同样是2天兑奖)经过一番沟通后，这位二十多岁的小伙子欣然接受记者专访，与媒体的距离就只隔一张薄薄的口罩。

　　号码不是机选守半年

　　“我这注号码不是机选的，销售员记错了，是守了半年的号，按上期的票照打的。”这是小伙子见到记者时第一句说的话。小伙子透露，自己买双色球[微博]已经有近四年时间，去年随家人从老家到广州工作，这个习惯也没有改变。小伙基本是期期都坚持投注，不过今年春节回老家则有近一个月的时间没有买彩，记者惊叹说幸好这注号码不是他回老家那个月开出，他反倒淡然地说那也是命中注定。

　　怕分薄奖金专选连号

　　去年刚到广州时，他守的是1注蓝球为“04”的单式号码，多数倍投10倍。去年8月，有一次打牌赢了钱，便豪气地想将倍数增至10倍，临投注前，他想着之前守的号码近半年没中出什么奖，于是干脆又另外写了1注，将蓝球改成“08”，叫销售员打了20倍，之后又让销售员机选了5注单式，

　　共投入50元，此后每期都按这两张彩票照打，结果就是这注自选的号码为他带来了亿元巨奖。

　　“这注号码有‘16、17、18’这组3连号，又有‘24、25’这组2连号，一般人自选很少这样选号?”面对记者的问题，小伙说自己就是特意选连号的，“就是越少人选的怪号，中出来注数才少，这样奖金才不会被分薄啊!”看来小伙中亿元巨奖的秘诀除了坚持守号外，就是专选怪号另辟蹊径了。

　　彩票放枕下整夜无眠

　　第13025期双色球周二开奖当晚，小伙并没有查看中奖号码，因此懵然不知自己已经成为亿万富翁。次日下午听同事说自己常去买彩的那家投注站敲锣打鼓庆祝中了亿元大奖，习惯倍投20倍的他就猜是不是自己中奖了?于是马上通过手机上网查看当期双色球开奖号码，终于确认中奖的正是自己!

　　小伙子马上将喜讯告诉了家人及女朋友，不过根本不敢说奖金有1亿元，只说自己中了大奖，后来姐姐、姐夫联系到夏茅中出1亿元大奖的新闻，才惊讶地发现弟弟中的居然是亿元巨奖。

　　小伙子说，他的彩票一直放在钱包里，昨晚就将钱包放在枕头底下，价值上亿的彩票就枕在自己头下，激动得整晚都睡不着觉。为免夜长梦多，次日下午他便拉上姐夫前往兑奖，成为最快现身的亿元巨奖得主。

　　拿到奖金先回馈母校

　　突然收到1.08亿多元这份大礼，不管是谁都会一时不知所措。当问及小伙如何分配这笔奖金时，小伙用坚定的口吻说，肯定首先要回馈培育自己的母校，特别希望为家乡的小孩子提供一个好的学习环境，让他们可以好好读书。同时，小伙子表示会先选择继续工作，等以后想法成熟了，再自己出来创业。据了解，根据广东福彩兑奖程序，在15个工作日内即可取到奖金现金支票。

　　临离开前，小伙说，购买双色球已成为了自己生活中一个习惯，虽然已经中了那么大的奖，但还是会继续购买福利彩票，奉献自己一份爱心。就在兑奖前，他还在福彩中心楼下的投注站购买了1注单式号码，依然是自己选的号，不过却没有再倍投了，只花了2元。

　　现场花絮

　　墨镜搁桌上只戴口罩

　　为保护大奖得主隐私，接受记者采访前，广东省福彩中心向其提供了口罩墨镜，但这位二十出头的年轻人似乎淡定得很，只将口罩松垮垮的戴在脸上，墨镜则一直搁在桌子上没用。

　　不过据工作人员透露，小伙子一开始其实紧张得很，填资料时脸上不停冒汗，手也抖得厉害，差点要靠一旁的姐夫帮忙才能填完资料，后来经过与工作人员一番闲聊，他的情绪才慢慢放松下来，更欣然接受媒体记者采访。

　　摆乌龙直接到省中心

　　原来，此次大奖得主兑奖还摆了一回乌龙。昨日下午2时，广东省福彩中心兑奖处刚刚上班，就有两名男子走了进来。其中，年轻的小伙从钱包中掏出彩票，工作人员一看，哇，亿元巨奖得主现身兑奖了!不过，工作人员发现，原来，小伙不清楚广东省福彩大奖的兑奖流程，没有先去广州市福彩中心验票，而是直接就杀到了省福彩中心。工作人员马上向二人详细介绍兑奖流程，大约四十分钟后，二人到位于西湖路的广州市福彩中心办完了验票手续后，才再次来到了广东省福彩中心兑奖。
